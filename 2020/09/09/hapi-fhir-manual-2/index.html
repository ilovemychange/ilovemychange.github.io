HAPI FHIR 환경 구성하기 (2/7편) - Eclipse 설치

오픈소스 HAPI FHIR 환경은 다음과 같이 7단계로 구성된다.
본 글에서는 2. HAPI FHIR 환경 구성하기 (2/7편) - Eclipse 설치에 대해 다룬다.

  1. HAPI FHIR 환경 구성하기 (1/7편) - JDK 설치
  2. HAPI FHIR 환경 구성하기 (2/7편) - Eclipse 설치
  3. HAPI FHIR 환경 구성하기 (3/7편) - Tomcat 설치
  4. HAPI FHIR 환경 구성하기 (4/7편) - MySQL 설치 및 스키마/계정 설정
  5. HAPI FHIR 환경 구성하기 (5/7편) - hapi-fhir-jpa-server 환경세팅
  6. HAPI FHIR 환경 구성하기 (6/7편) - hapi-fhir 테스트 페이지 활용
  7. HAPI FHIR 환경 구성하기 (7/7편) - hapi-fhir-client 환경세팅

Eclipse란, Java를 기반으로 한 오픈소스 IDE(통합개발환경)이다.

  1. 이클립스 다운로드 페이지로 접속하여, Download 64 bit 파일을 다운로드한다.

  1. 다운로드 버튼을 클릭하여 Eclipse Installer를 내려받는다.

  1. 다운로드받은 Eclipse Installer를 실행한다. Installer를 통해 Eclipse의 사용자에게 필요한 패키지를 선택하여 설치할 수 있다. Eclipse IDE for Enterprise Java Developers를 선택하여 진행한다.

  1. JVM 경로와 설치경로(Installation Folder)를 지정한다. JVM은 앞서 설치한 jdk 폴더 경로를 지정하고, 설치경로는 프로젝트 폴더(D:\hapi-fhir)로 지정한다.

  1. 소프트웨어 사용과 관련된 동의서이다. Accept Now를 누르고 진행한다.

  1. 설치가 진행된다. 중간중간 팝업창이 뜨면 계속 진행하면 된다.

  1. 설치가 완료되었다.

  1. 지정한 설치경로 (D:/hapi-fhir) 아래에 다음과 같이 eclipse 폴더가 생성이 된다.

Eclipse 설치는 이것으로 마무리한다.

끝.
